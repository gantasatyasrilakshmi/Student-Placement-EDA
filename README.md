# Student Placement EDA

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a Student Placement dataset using Python. The objective is to understand the factors that influence student placements through statistical analysis and visualizations.

## Dataset

- Dataset: Student Placement Synthetic Dataset
- Records: 10,000
- Format: CSV

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

- Data Loading
- Data Cleaning
- Data Exploration
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Data Visualization
- Key Insights

## Key Insights

- Placement rates vary with academic performance.
- Internship experience positively influences placement.
- Communication skills show a strong relationship with placement outcomes.
- Multiple visualizations were created to identify important trends.
